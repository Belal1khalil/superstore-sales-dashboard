# Superstore Sales & Profit Analysis Dashboard

## Overview
Interactive Excel dashboard analyzing 5,000+ orders to uncover sales, 
profit, and regional performance trends.

## Key Insights
- Total Sales: $2.29M | Total Profit: $286K
- YoY Growth: 51.41%
- Profit Margin: 12.47%
- West & East regions are top performers
- Technology category drives highest profitability

## Tools & Techniques
- **Power Query**: Data cleaning, transformation, merging tables
- **Power Pivot / DAX**: Custom measures (Total Sales, Profit Margin, YoY Growth, etc.)
- **PivotTables/Charts**: Dynamic, slicer-driven visuals
- **Custom theming**: Slicer styles, color palettes for readability
